# TikTok Ad Toolkit — 使用说明

> Vidau 自研 TikTok 广告平台智能工具集 v1.0

## 快速开始

### 1. 系统要求

- **运行平台**: Hermes Agent 运行时环境
- **目标网站**: https://tiktok.vidau.ai/
- **浏览器支持**: Chrome (推荐) / Chromium

### 2. 安装步骤

1. 登录 [Vidau TikTok 智能投放平台](https://tiktok.vidau.ai/)
2. 进入 Hermes Skill 管理页面
3. 上传本工具集的根目录（即包含此 README.md 的文件夹）
4. 等待系统解析并激活所有子模块
5. 在 **AI助手** 页面即可开始使用

### 3. 功能概览

#### 🎯 skills/creation — 广告创建
- 支持闭环广告创建工作流
- 自动校验必填字段（账户→广告系列→广告组→广告→素材）
- 创建失败自动重试（最多3次，仅限可重试错误）
- 成功后可选开启预警中心监控

#### 🔍 skills/inspection — 数据巡检
- 8 大异常检测维度（花费、转化、CPA、ROAS、预算、素材疲劳等）
- 基于真实数据的异常诊断和优先级排序
- 输出建议但不执行操作（recommendation_only）
- 支持每日自动巡检询问（不自动创建定时任务）

#### 📚 skills/knowledge — 知识问答
- 自然语言意图识别与模块路由
- 支持 10 大业务模块（开户、授权、同步、报表、素材、搭建、预警等）
- 统一的 AI 助手响应格式（JSON + assistant_display）
- 完整的字位槽提取和缺失项处理

#### 📊 skills/report — 投放报告
- 自动生成日报/周报（Markdown 卡片格式）
- 智能账户筛选（已授权且余额 > 0）
- 完整的报告结构：核心数据、环比变化、问题/优秀广告、优化建议、风险提醒
- AI 助手展示失败时降级到当前对话界面（需尝试10次）

## 触发关键词速查

| 意图 | 触发词示例 | 对应模块 |
|------|-----------|----------|
| 创建广告 | 创建TikTok广告、新建广告计划、打开创建页面 | creation |
| 数据查看 | 查看数据、投放数据、数据分析、数据巡检 | inspection |
| 知识问答 | 怎么开户、如何授权、广告层级是什么 | knowledge |
| 生成报告 | 生成日报、生成周报、看下昨天数据 | report |

## 注意事项

1. **权限要求**: 所有操作需要 TikTok 广告账户已在 Vidau 平台完成授权
2. **余额检查**: 默认只处理余额 > 0 的账户
3. **数据同步**: 执行分析前确保已完成最新数据同步
4. **确认机制**: 高风险操作会弹出确认提示，请仔细核对后再执行
