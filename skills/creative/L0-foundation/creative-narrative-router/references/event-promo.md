# event_promo — Sale / launch promo

> ID: `event_promo`  
> Use when: offer, discount, launch, seasonal sale — **conversion-led**.

## When to use

- “Sale”, “launch”, “Black Friday”, “limited time”, “% off”, “bundle”
- Explicit offer mechanics in brief
- 15–45s (dense info OK)

## Beat map — 30s

| Beat | Duration | Visual |
|------|----------|--------|
| `hook_offer` | 0–3s | Biggest benefit visible immediately |
| `urgency` | 3–8s | Time/scarcity **only if user provided** |
| `benefit_stack` | 8–20s | 2–3 stacked benefits |
| `mechanism` | 20–26s | How to redeem |
| `cta` | 26–30s | Strong CTA + brand |

## 15s ultra-short

`hook_offer` → `benefit_stack` → `cta` (3 shots).

## Scene types

`offer_card` · `price_highlight` · `countdown` · `product_grid` · `cta_burst`

## Forbidden

- Fake countdown / fake “only 3 left” without user data
- Hidden terms
- No offer in first 5s

## Voiceover

30s: ~55–75 words, punchy.
