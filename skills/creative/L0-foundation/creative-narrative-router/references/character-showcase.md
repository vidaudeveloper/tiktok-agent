# character_showcase — Character / talent showcase

> ID: `character_showcase`  
> Use when: pose, expression, motion, lookbook — **not product-led**.

## When to use

- “Character showcase”, “model”, “lookbook”, “turnaround”, “casting reel”
- User wants person/avatar focus, not SKU sell
- Aligns with vidau `character_showcase` ad format

## Beat map — 30s

| Beat | Duration | Visual |
|------|----------|--------|
| `intro_pose` | 0–6s | Full look / signature pose |
| `expression` | 6–14s | Face / emotion |
| `action` | 14–22s | Gesture / movement |
| `environment` | 22–27s | Character in space |
| `closing_portrait` | 27–30s | Portrait hold |

## Scene types

`portrait` · `full_body` · `expression_closeup` · `motion_gesture`

## Forbidden

- Rewriting as product ad
- Invented gadgets (watch UI, phone specs) user never mentioned
- Product hero shots unless user added product refs

## Narrative Driver

Lead with **look, expression, motion**; BGM + ambient sound over spec VO.

## Voiceover

≤40 words / 30s or none.
