# knowledge_explainer — Educational / explainer

> ID: `knowledge_explainer`  
> Use when: teach a concept, how-it-works, tutorial tone.

## When to use

- “Explainer”, “how X works”, “what is”, “tutorial”, “educational”
- Progressive understanding (works well with `voiceover: true`)
- 30–120s

## Beat map — 60s

| Beat | Duration | Role |
|------|----------|------|
| `hook_question` | 0–5s | Question or surprise — not “In this video…” |
| `setup_gap` | 5–15s | Why viewer should care |
| `build_steps` | 15–45s | 2–4 steps; **therefore/but** links |
| `aha` | 45–55s | Insight connects |
| `landing` | 55–60s | One-line recap (+ optional CTA) |

Compress for 30s: merge setup into hook; 2 build steps only.

## Scene types

`question_card` · `diagram` · `step_demo` · `summary_card`

## Forbidden

- “In this video we'll learn about…”
- Random examples off-topic
- Skipping logical steps

## Voiceover (~150 wpm)

| Length | English words |
|--------|---------------|
| 30s | ~50–70 |
| 60s | ~100–130 |
| 90s | ~150–200 |

## Narrative Driver

State the question promised in hook + logic chain + aha payoff.
